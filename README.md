# Games with Friends

This repo creates a server that manages a phone based trivia games. This server hosts the game and users use their phones to answer questions. Similar to Jackbox games, there is one main screen that is displayed on a TV and the players use their phones to answer questions. 

## Getting started

Install uv

```bash
brew install uv
```

Update dependencies
```bash
uv sync
```

Run server
```bash
uv run main.py
```
